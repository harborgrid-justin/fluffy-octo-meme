## Introduction

The Federal PPBE Management System is a comprehensive application designed to support the full Planning, Programming, Budgeting, and Execution (PPBE) lifecycle for US federal government agencies.

### What is PPBE?

PPBE is the primary resource allocation process used by the Department of Defense and other federal agencies:

- **Planning**: Define strategic objectives and identify required capabilities
- **Programming**: Allocate resources to programs and initiatives
- **Budgeting**: Develop detailed budget requests and justifications
- **Execution**: Monitor spending and track program performance

## Getting Started

### Accessing the System

1. Open your web browser
2. Navigate to the system URL (e.g., https://ppbe.your-agency.gov)
3. You will be presented with the login page

### Logging In

**Default Credentials** (Development Only):
- Username: `admin`
- Password: `admin123`

**Production**: Use your assigned government credentials

1. Enter your username
2. Enter your password
3. Click the "Login" button

**Security Note**: After 5 failed login attempts, you will be temporarily locked out for 15 minutes.

### Navigation

Once logged in, you'll see four main sections in the navigation bar:

1. **Dashboard** - Overview and key metrics
2. **Budget Management** - Create and manage budgets
3. **Program Planning** - Define and track programs
4. **Execution Tracking** - Monitor spending and progress

## Dashboard

The Dashboard provides a high-level overview of your agency's PPBE activities.

### Key Metrics

**Total Budget**: The sum of all budget allocations across fiscal years
- Shows overall budget authority
- Updates in real-time as budgets are added or modified

**Active Programs**: Number of programs currently in execution
- Displays active programs vs. total programs
- Quick indicator of workload

**Budget Executed**: Total amount spent across all programs
- Shows actual expenditures
- Includes utilization percentage

**Pending Approvals**: Number of budget items awaiting approval
- Action items requiring attention
- Click to filter and view pending items

### Budget by Department

Visual breakdown showing:
- Budget allocation per department
- Percentage of total budget
- Progress bars for easy comparison

### System Status

Displays:
- System operational status
- Number of active departments
- Last update timestamp

## Budget Management

Create, view, edit, and delete budget allocations.

### Creating a Budget

1. Click the "**+ New Budget**" button
2. Fill in the required fields:
   - **Title**: Descriptive name for the budget (e.g., "FY2025 Operations Budget")
   - **Fiscal Year**: Select the applicable fiscal year
   - **Department**: Choose the responsible department
   - **Amount**: Enter the budget amount in dollars
   - **Category**: Select budget category (Operations, Personnel, Equipment, etc.)
   - **Status**: Set status (Draft, Pending, Approved)
   - **Description**: Optional notes and justification

3. Click "**Create Budget**"

### Viewing Budgets

All budgets are displayed in a table showing:
- Title and fiscal year
- Department and amount
- Category and status
- Creator information
- Action buttons

### Filtering Budgets

Use the filter dropdowns to narrow results:
- **Fiscal Year**: Filter by specific year
- **Department**: Show only specific department budgets
- **Status**: Filter by approval status

### Editing a Budget

1. Click the "**Edit**" button on the budget row
2. Modify the fields as needed
3. Click "**Update Budget**"

### Deleting a Budget

1. Click the "**Delete**" button on the budget row
2. Confirm the deletion in the popup dialog

**Warning**: Deletion is permanent and cannot be undone.

### Budget Categories

- **Operations**: Day-to-day operational expenses
- **Personnel**: Salaries, benefits, training
- **Equipment**: Capital purchases, hardware, software
- **Infrastructure**: Facilities, maintenance, utilities
- **Research & Development**: Innovation and R&D initiatives

## Program Planning

Define and manage federal programs and initiatives.

### Creating a Program

1. Click "**+ New Program**"
2. Fill in the required information:
   - **Program Name**: Clear, descriptive name
   - **Fiscal Year**: Program execution year
   - **Department**: Owning department
   - **Budget**: Total program budget
   - **Priority**: Low, Medium, or High
   - **Start Date**: Program start date
   - **End Date**: Expected completion date
   - **Status**: Planning, Active, or Completed
   - **Objectives**: Program goals and objectives
   - **Key Milestones**: Major deliverables and dates

3. Click "**Create Program**"

### Program Lifecycle

Programs move through different statuses:

1. **Planning**: Initial development and approval
2. **Active**: Currently executing
3. **Completed**: Finished or closed out

### Managing Programs

- **Edit**: Update program details as plans evolve
- **Delete**: Remove programs (with confirmation)
- **Filter**: Search by fiscal year, department, or status

### Priority Levels

- **High**: Critical, mission-essential programs
- **Medium**: Important but not critical
- **Low**: Nice-to-have, future consideration

## Execution Tracking

Monitor program execution and spending.

### Creating an Execution Record

1. Click "**+ New Record**"
2. Complete the form:
   - **Program**: Select the program (required)
   - **Fiscal Year**: Execution fiscal year
   - **Amount Spent**: Actual expenditure
   - **Status**: On Track, At Risk, Delayed, or Completed
   - **Execution Date**: Date of expenditure
   - **Description**: What was purchased/accomplished
   - **Notes**: Additional context or issues

3. Click "**Create Record**"

### Execution Status

**On Track**: Progressing as planned
- Green indicator
- Meeting milestones and budget

**At Risk**: Potential issues identified
- Yellow indicator
- May need intervention

**Delayed**: Behind schedule
- Red indicator
- Requires corrective action

**Completed**: Execution finished
- Gray indicator
- Closed out

### Tracking Spending

The execution records provide:
- Historical spending data
- Budget vs. actual comparison
- Trend analysis over time
- Early warning of overruns

## Common Tasks

### Creating a Complete Budget Package

1. Create budget allocations for each department
2. Set initial status to "Draft"
3. Add detailed descriptions and justifications
4. Review and adjust amounts
5. Change status to "Pending" when ready for approval
6. After approval, update status to "Approved"

### Monitoring Program Performance

1. Navigate to Execution Tracking
2. Filter by the specific program
3. Review all execution records
4. Check for status indicators (at-risk or delayed)
5. Compare total spent vs. program budget
6. Take corrective action as needed

### Preparing Budget Reports

1. Go to Dashboard
2. Review Budget by Department section
3. Use filters in Budget Management to drill down
4. Export data (feature can be added) or take screenshots
5. Analyze budget utilization percentages

### End-of-Year Close Out

1. Ensure all execution records are entered
2. Update program statuses to Completed
3. Verify budget vs. actual reconciliation
4. Document lessons learned in program notes
5. Prepare data for next fiscal year

## Tips and Best Practices

### Data Entry

- Use consistent naming conventions
- Include detailed descriptions for future reference
- Update records regularly, not just at end of period
- Double-check amounts before saving
- Use proper fiscal year designations

### Security

- Log out when finished using the system
- Don't share credentials
- Use strong, unique passwords
- Report suspicious activity immediately
- Follow agency security policies

### Budget Planning

- Start with historical data
- Include contingency buffers
- Align with strategic objectives
- Coordinate across departments
- Document assumptions and justifications

### Program Management

- Define clear, measurable objectives
- Set realistic timelines
- Assign appropriate priority levels
- Update status regularly
- Track milestones and deliverables

### Execution Tracking

- Enter expenditures promptly
- Include detailed descriptions
- Monitor burn rate vs. timeline
- Flag issues early
- Document corrective actions

## Troubleshooting

### Can't Log In

**Problem**: Login fails with correct credentials
**Solution**: 
- Verify Caps Lock is off
- Check for extra spaces
- Wait 15 minutes if locked out after failed attempts
- Contact system administrator if issues persist

### Data Not Saving

**Problem**: Changes don't appear after saving
**Solution**:
- Check for error messages at top of screen
- Ensure all required fields are completed
- Verify network connection
- Refresh the page and try again
- Check with IT if problem continues

### Missing Data

**Problem**: Can't find previously entered data
**Solution**:
- Check filter settings (may be hiding data)
- Verify you're looking in correct fiscal year
- Clear all filters and search again
- Data may have been deleted by another user

### Slow Performance

**Problem**: System is responding slowly
**Solution**:
- Check your internet connection
- Close unnecessary browser tabs
- Clear browser cache
- Try a different browser
- Contact IT if problem affects multiple users

### Budget Not Appearing on Dashboard

**Problem**: New budget doesn't show in dashboard totals
**Solution**:
- Refresh the page (F5 or Ctrl+R)
- Verify the budget was saved (check Budget Management)
- Check that fiscal year is correct
- Dashboard updates in real-time but may need refresh
